# SmoothFenceUSA – Project Status

## 1. Overview

Modern web platform for a fence installation & repair business in Florida.

Goals:
- High-conversion landing page (leads → sales).
- Gallery of finished fence projects (full-screen view).
- Simple quote flow (lead capture and basic price estimation).
- Future: payments online, admin panel, IA assistant + n8n automations.

Tech stack:
- Next.js (App Router)
- TypeScript
- Tailwind CSS
- Framer Motion (animations)
- PostgreSQL + Prisma
- Stripe (payments – future)
- n8n / automations (future)
- AI assistant integration (future)
