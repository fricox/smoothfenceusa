# Project Context: SmoothFenceUSA

Business: SmoothFenceUSA – fence installation business located in Florida, USA.  
Target audience: Homeowners and property managers looking for vinyl, aluminum, chain-link and wood fences.  
Goals of the website:
- Generate leads (quote requests, contact form submissions).
- Build trust with local customers (gallery, reviews, HOA/permits info).
- Rank on Google for terms like “fence installation”, “vinyl fence”, “aluminum fence”, “Palm Coast fence contractor”, etc.

Tech stack:
- Next.js (App Router) + TypeScript
- Tailwind CSS
- Deployed to production (Vercel or suitable hosting)
- Future integration: forms (quote requests), basic analytics, SEO.

General rules for any AI agent:
- Always keep the brand name: "SmoothFenceUSA" (no space variations).
- Write copy in US English, clear and persuasive, oriented to homeowners.
- Keep code clean, typed (TypeScript), and split into reusable components.
- Prefer server components where possible, but use client components for interactivity (forms, animations, etc.).
- Avoid magic values: use config/constants when helpful.
- Keep SEO in mind: proper `<title>`, meta description, headings, alt text for images.
- Never hardcode secrets in the repo.
